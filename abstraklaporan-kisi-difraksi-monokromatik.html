1. Percobaan penelitian difraksi cahaya monokromatik menggunakan kisi dengan 2.000 garis/mm. Tetapan/konstanta kisi tersebut adalahdengan caranya ya kaa​


Penjelasan:

Tetapan/konstanta kisi dapat dicari dengan menggunakan rumus:

d sin θ = mλ

dalam rumus tersebut, d adalah jarak antar garis pada kisi dalam satuan meter, θ adalah sudut dari sinar difraksi, m adalah urutan bayangan, dan λ adalah panjang gelombang cahaya monokromatik dalam satuan meter.

Kita sudah diberikan informasi bahwa kisi tersebut memiliki 2.000 garis/mm, sehingga jarak antar garis (d) dapat dicari dengan:

d = 1/2000 mm = 0,0005 mm = 5 x 10^-7 m

Kita juga sudah diberikan informasi bahwa percobaan menggunakan cahaya monokromatik, sehingga panjang gelombang (λ) juga diketahui.

Untuk mencari sudut difraksi (θ) yang pertama, kita dapat menggunakan rumus:

sin θ = mλ/d

Kita akan menggunakan urutan bayangan m = 1 (bayangan pertama), sehingga:

sin θ = 1λ/d = 1 x λ/(5 x 10^-7 m)

Jika sudut θ sangat kecil, maka sin θ dapat dianggap sama dengan θ dalam radian. Menggunakan kalkulator, kita dapat menghitung nilai θ:

θ = sin^-1(λ/d) = sin^-1(1 x λ/(5 x 10^-7 m)) = sin^-1(λ/0,0005)

Jadi, tetapan/konstanta kisi untuk kasus ini adalah:

d sin θ = (5 x 10^-7 m) sin[sin^-1(λ/0,0005)] = λ

Sehingga, tetapan/konstanta kisi pada percobaan ini adalah sama dengan panjang gelombang cahaya monokromatik yang digunakan.

Garis-garis gelap dan terang yang terjadi pada layar ketika seberkas cahaya monokromatik melalui kisi dapat diterangkan dengan b. Difraksi dan Interferensi

PEMBAHASAN

Difraksi cahaya adalah peristiwa pembelokan arah rambat cahaya karena melewati celah sempit.

Kisi difraksi adalah peristiwa difraksi saat cahaya melewati celah-celah sempit yang sangat halus dan banyak.

d. sin Ф = n . λ

dimana:

d = jarak antar celah sempit ( m )

Ф = sudut pembelokan cahaya ( derajat )

n = orde

λ = panjang gelombang ( m )

Jika cahaya mengenai suatu celah tunggal maka berlaku rumus yang mirip yakni :

d. sin Ф = n . λ

dimana:

d = lebar celah tunggal ( m )

Ф = sudut pembelokan cahaya ( derajat )

n = orde

λ = panjang gelombang ( m )

Mari sekarang kita mencoba menyelesaikan soalnya.

Saat cahaya masuk melalui kisi atau celah sempit maka cahaya akan mengalami pembelokan cahaya atau di kenal dengan istilah Difraksi.

Selanjutnya Berkas-berkas cahaya yang keluar dari celah-celah sempit ini akan mengalami penggabungan atau dikenal dengan istilah Interferensi.

Interferensi yang terjadi terdiri dari :

Interferensi Konstruktif dimana berkas-berkas cahaya bergabung dan saling menguatkan sehingga terbentuk pita terang di layarInterferensi Destruktif dimana berkas-berkas cahaya bergabung dan saling melemahkan sehingga terbentuk pita gelap di layar

Kesimpulan :

Seberkas cahaya monokromatik melalui kisi dapat diterangkan dengan Difraksi dan Interferensi

Keterangan:  

p = jarak pita terang ke-m  

L = jarak celah ke layar

Hanya definisi ( d ) dalam persamaan memiliki pengertian yang berbeda :

Difraksi tunggal : d = diameter celah  

Kisi Difraksi : d = jarak antar celah yang nilainya = 1/N

(1)Interferensi Young ( Celah Ganda )  

d sin θ = Δm λ

d p = Δm L λ

(2) Kisi Difraksi  

d sin θ = Δm λ

d p = Δm L λ

d = 1/N  

Untuk mempermudah dalam menentukan nilai Δm dalam penyelesaian soal, maka pada interferensi ganda maupun kisi difraksi dapat menggunakan aturan berikut :  

terang ke terang yang berdekatan (Δm = 1 )  

gelap ke gelap yang berdekatan (Δm = 1 )  

 terang ke gelap yang berdekatan (Δm = ½ )

( 3 ) Difraksi Tunggal  

d sin θ = m λ

d p = m L λ

16. Kisi difraksi mempunyai 4000 goresan tiap cm. Kisi tersebut didatangkan cahaya monokromatik dan menghasilkan garis terang orde ke-2. Apabila sudut deviasinnya 30°, maka panjang gelombang cahaya adalah... (1A°=10^-10m)


Mata pelajaran : Fisika

Kelas: XI SMA

Kategori : gelombang cahaya/ optika fisis

Kata kunci : kisi difraksi, interferensi cahaya, celah tunggal, celah ganda, percobaan young, panjang gelombang, orde, terang, gelap, jarak layar, sudut difraksi, optika fisis

Kode kategori berdasarkan kurikulum KTSP : 12.6.3

 

 

JAWABAN :

 

Dari soal dapat diketahui bahwa:

goresan tiap cm --> N = 4000

garis terang orde ke-2 --> n = 2

sudut deviasinya --> θ = 30

 

Ditanyakan panjang gelombang cahaya --> λ

 

Jawab:

Gunakan rumus kisi difraksi dengan sudut deviasi. Mula- mula dicari d.

4000 goresan tiap cm diubah menjadi 400.000 goresan tiap m, sehingga:

d = 1/N = 1/400.000

d = 0,0000025 m

 

sehingga untuk mencari panjang gelombang cahaya

nλ = d sin θ

2. λ = 0,0000025. Sin 30

λ = 0,0000025. ½. ½

λ = 0,000000625 m atau

λ = 6250 angstrom

 

Jadi panjang gelombang cahayanya sebesar 6250 angstrom.

 

 

PEMBAHASAN LEBIH LANJUT :

 

Persamaan umum yang digunakan pada interferensi dan difraksi cahaya yaitu:

nλ = d.p/l

 

atau

nλ = d sin θ

 

keterangan:

n = orde terang/ gelap

λ = panjang gelombang

d = jarak celah

p = jarak antar pola

l = jarak layar ke celah

θ = sudut deviasi

 

A.      Interferensi celah ganda/ Young

Pola terang: n = 1, 2, 3, 4, dst..

Pola gelap: n = ½, 3/2, 5/2, 7/2, dst.

 

B.       Difraksi celah tunggal

Pola terang: n = 3/2, 5/2, 7/2, 9/2, dst.

Pola gelap: n = 1, 2, 3, 4, dst..

 

C.       Difraksi kisi

Pada kisi difraksi terdapat jumlah celah/ goresan tiap meter yang disebut N

N = 1/d

25. Kisi difraksi memiliki 20.000 garis/cm. sinar monokromatik di lewatkan pada kisi sehingga terbentuk terang kedua pada jjarak 4,2 mm. sudut yang terbentuk saat terang kedua adalah 30o. Panjang gelombang yang digunakan sebesar


Difraksi cahaya merupakan peristiwa pelenturan cahaya karena melalui celah kecil. Pada difraksi kisi, cahaya dilenturkan melalui celah- celah kecil berjumlah banyak sehingga akan tampak pola terang denga orde tertentu.


Dari soal di atas dapat diketahui bahwa:

- Terjadi difraksi kisi/ celah majemuk

- sudut difraksi orde dua --> θ = 30

- konstanta kisi --> N = 20.000 garis/ cm = 2.000.000 garis/ m

 

Sedangkan yang menjadi permasalahan pada soal adalah panjang gelombang.

 

Mula – mula kita cari n untuk terang orde dua yaitu n = 2.

Kemudian kita cari dahulu d:

d = 1/N = 1/ 2.000.000 = 5. 10^{-7}

 

Untuk mencari panjang gelombang digunakan rumus berikut: 

nλ = d sin θ

2. λ = 5. 10^{-7}. Sin 30

λ = 1,25. 10^{-7} m = 125. 10^{-9} nm = 125 nm

 

Berdasarkan uraian di atas, didapat bahwa panjang gelombang yang digunakan 125 nm.

Ketentuan n pada kisi difraksi:

Pola gelap: n = 1, 2, 3, 4, dst..
