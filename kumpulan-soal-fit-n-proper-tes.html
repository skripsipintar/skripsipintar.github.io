2. jelaskan istilah fit and proper test


Fit and proper test atau uji kepatutan dan kelayakan merupakan salah satu istilah yang kerap kita dengar dalam kehidupan sehari-hari. Istilah ini biasanya dipergunakan oleh berbagai lembaga sebelum memilih orang yang tepat dalam menduduki suatu jabatan. Salah satu lembaga negara yang kerap menggunakan proses ini adalah lembaga legislasi kita. Fit dan proper test yang dilakukan biasanya terdiri dari beberapa bagian. Langkah-langkah tersebut antara lain mencakup pemeriksaan riwayat hidup, uji tertulis, hingga wawancara. Semua tes tersebut dimaksudkan untuk mendapatkan kandidat yang terbaik.


Pada kesempatan ini, soal meminta kita untuk menjelaskan istilah fit and proper test. Selain dari pengertian di atas, berikut kakak akan mencoba menjelaskan istilah tersebut dengan lebih ringkas.

8. Ani telah mengikuti tes matematika sebanyak n kali. Pada tes berikutnya, ia memperoleh nilai 83 sehingga nilai rata-rata Ani adalah 80. Tetapi jika tes tersebut adalah 67, maka rata-ratanya adalah 76. Nilai n adalah...


Penjelasan:

Pada saat n + 1 nilai Ani = 83

dan rata² nya (R) = 80

misalkan pada saat n kali jumlah nilai ani = x maka jumlah nilai ani pada tes berikutnya

(n + 1) adalah (x + 83)

rata² nilai ani pada saat (n + 1)

R = (x + 83)/(n + 1)

80 = (x + 83)/(n + 1)

(n + 1) = (x + 83)/80..........(1)

jika pada saat tes tersebut/saat (n + 1) nilai ani = 67 dan rata²nya (R) = 76 maka

R = (x + 67)/(n + 1)

76 = (x + 67)/(n ,+ 1)

(n + 1) = (x + 67)/76..........(2)

subtitusikan (1) = (2)

(x + 83)/80 = (x + 67)/76

80 (x + 67) = 76 (x + 83)

80x + 5360 = 76x + 6308

80x - 76x = 6380 - 5360

4x = 948

x = 237.....nilai x masukan ke (1)

(n + 1) = (x + 83)/80

(n + 1) = (237 + 83)/80

(n + 1) 320/80

(n + 1) = 4

n = 4 - 1

n = 3

13. Ani telah mengikuti tes matematika sebanyak n kali. Pada tes berikutnya ia memperoleh nilai 83 sehingga nilai rata rata ani adalah 80. Tetapi jika nilai tes tersebut 67 maka rata ratanya adalah 76. Nilai n adalah....


Misal Nilai rata - rata sebelum tes berikutnya = X
1) jika mendapat nilai 83 rata-rata nya 80
(nX + 83)/(n + 1) = 80
=> nX + 83 = 80n + 80
=> nX = 80n - 3


2) jika mendapat nilai 67 rata-ratanya 76
(nX + 67)/(n + 1) = 76
=> nX + 67 = 76n + 76
=> nX = 76n + 9

nX = nX
80n - 3 = 76n + 9
4n = 12
n = 3

40. ani telah mengikuti tes matematika sebanyak n kali. pada tes berikutnya ia mendapatkan nilai 83 sehingga nilai Rata2 ani adalah 80. tetapi, jika nilai tes tsb adalah 67, maka rata2nya Adalah 76. nilai n adalah


Ani telah mengikuti tes matematika sebanyak n kali. Pada tes berikutnya ia mendapatkan nilai 83 dengan nilai rata rata adalah 80. Tetapi, jika nilai tesnya adalah 67, maka nilai rata rata nya adalah 76. Maka nilai n adalah 3.

Penyelesaian Soal :

LANGKAH PERTAMA (I)

Buatlah persamaan dari pernyataan "Ani telah mengikuti tes matematika sebanyak n kali. Pada tes berikutnya ia mendapatkan nilai 83 dengan nilai rata rata adalah 80" yaitu :

X = ∑X/ n

Keterangan : X = Nilai rata rata  

                      ∑X = Jumlah nilai

                      n = banyaknya data

Maka perhitungannya yaitu :

X = ∑X/ n

80 = (x + 83) / (n + 1)

80 (n + 1) = x + 83

80n + 80 = x + 83

x - 80n = 80 - 83

x - 80n = -3      ... (Persamaan 1)

LANGKAH KEDUA (II)

Buatlah persamaan dari pernyataan "jika nilai tesnya adalah 67, maka nilai rata rata nya adalah 76" yaitu :

X = ∑X/ n

76 = (x + 67) / (n + 1)

76 (n + 1) = x + 67

76n + 76 = x + 67

x - 76n = 76 - 67

x - 76n = 9       ... (Persamaan 2)

LANGKAH KETIGA (III)

Eliminasikan persamaan 1 dan persamaan 2 sehingga diperoleh nilai n dengan menggunakan cara sebagai berikut :

x - 80n = -3

x - 76n = 9

___________ -

   -4n = -12

      n = [tex]\frac{12}{4}[/tex]

      n = 3

∴ Kesimpulan nilai n adalah 3.
