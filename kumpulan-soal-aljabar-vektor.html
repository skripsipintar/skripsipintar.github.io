Jawaban:

Bentuk aljabar adalah teknik yang digunakan untuk menyajikan suatu masalah Matematika dengan simbol atau huruf sebagai peubah suatu objek dalam masalah tersebut. Bentuk aljabar terdiri dari konstanta (nilai tetap) dan variabel (nilai peubah) melalui operasi penjumlahan, pengurangan, perkalian, pembagian, perpangkatan, dan pengakaran.

contoh soal :

Berapakah hasil pemfaktoran dari bilangan berikut ini 16x2 − 9y2 ?

Jawabannya :

Harus di ingat bahwa bentuk faktor nya yakni aljabar :

a2 – b2 = ( a + b ) ( a – b )

16x2 = ( 4x )2

9y2 = ( 3y )2

Sehingga faktor dari bilangan 4x2 – 9y2 ialah ?

16x2 – 9y2 = ( 4x + 3y ) ( 4x – 3y )

Jadi, hasil pemfaktoran dari bilangan 16x2 − 9y2 ialah : ( 4x + 3y ) ( 4x – 3y).

a) A⋅ B
b) A × B

Pembahasan
a) A⋅ B adalah perkalian titik (dot) antara vektor A dan vektor B
Untuk perkalian titik berlaku
A⋅ B = A B cos θ 
Sehingga
A⋅ B = A B cos 37° = (8)(10)(0,8) = 64 satuan

b) A × B adalah perkalian silang (cross) vektor A dan vektor B
Untuk perkalian silang berlaku
A × B = A B sin θ 
Sehingga
A × B = A B sin 37° = (8)(10)(0,6) = 48 satuanSoal No. 11
