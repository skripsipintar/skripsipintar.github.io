13. Dua gelombang cahaya diarahkan tegak lurus pada celah ganda jika cahaya pertama memiliki panjang gelombang 600 nm, berapa panjang gelombang cahaya lainnya agar garis terang kedua dari cahaya pertama berimpit dengan garis terang keempat dari cahaya lainnya


Mata pelajaran : Fisika

Kelas: XI SMA

Kategori : gelombang cahaya/ optika fisis

Kata kunci : kisi difraksi, interferensi cahaya, celah tunggal, celah ganda, percobaan young, panjang gelombang, orde, terang, gelap, jarak layar, sudut difraksi, optika fisis

Kode kategori berdasarkan kurikulum KTSP : 12.6.3

 

 

JAWABAN :

 

Dari soal dapat diketahui bahwa:

panjang gelombang --> λ1 = 600 nm

garis terang orde ke-2 --> n1 = 2

garis terang orde ke-4 --> n2 = 4

 

Ditanyakan panjang gelombang cahaya lainnya --> λ2

 

Jawab:

Agar garis terang kedua dari cahaya pertama berimpit dengan garis terang keempat dari cahaya lainnya, panjang gelombang cahaya lainnya dicari dengan membandingkan λ dan n saja.

 

nλ = d.p/l

dari rumus di atas, kita lihat bahwa λ berbanding terbalik dengan n, sehingga

λ1/ λ2 = n2/n1

600/ λ2 = 4/2

λ2 = 600/2 = 300 nm

 

Jadi panjang gelombang cahaya lainnya 300 nm.

 

 

PEMBAHASAN LEBIH LANJUT :

 

Persamaan umum yang digunakan pada interferensi dan difraksi cahaya yaitu:

nλ = d.p/l

 

atau

nλ = d sin θ

 

keterangan:

n = orde terang/ gelap

λ = panjang gelombang

d = jarak celah

p = jarak antar pola

l = jarak layar ke celah

θ = sudut deviasi

 

A.      Interferensi celah ganda/ Young

Pola terang: n = 1, 2, 3, 4, dst..

Pola gelap: n = ½, 3/2, 5/2, 7/2, dst.

 

B.       Difraksi celah tunggal

Pola terang: n = 3/2, 5/2, 7/2, 9/2, dst.

Pola gelap: n = 1, 2, 3, 4, dst..

 

C.       Difraksi kisi

Pada kisi difraksi terdapat jumlah celah/ goresan tiap meter yang disebut N

N = 1/d

35. seberkas cahaya monokromatik dengan panjang gelombang 600 nm menyinari tegak lurus kisi yg memiliki tetapan 300 garis/mm.tentukan orde maksimum yg dapat di amati

PEMBAHASAN

Difraksi cahaya adalah peristiwa pembelokan arah rambat cahaya karena melewati celah sempit.

Kisi difraksi adalah peristiwa difraksi saat cahaya melewati celah-celah sempit yang sangat halus dan banyak.

d. sin Ф = n . λ

dimana:

d = jarak antar celah sempit ( m )

Ф = sudut pembelokan cahaya ( derajat )

n = orde

λ = panjang gelombang ( m )

Mari sekarang kita mencoba menyelesaikan soalnya.

Diketahui:

λ = 600 nm = 6 . 10⁻⁷ m

d = 1/300 mm = (1/3) . 10⁻⁵ m

Ditanyakan:

n = ?

Penyelesaian:

d. sin Ф = n . λ

sin Ф = n . λ / d

Berhubung Ф < 90° , maka :

sin Ф < 1

n . λ / d  <  1

n  <  d / λ

n  <  [ (1/3) . 10⁻⁵ ]  / (6 . 10⁻⁷)

n < 5,6

Kesimpulan :

Orde maksimumnya adalah 5
