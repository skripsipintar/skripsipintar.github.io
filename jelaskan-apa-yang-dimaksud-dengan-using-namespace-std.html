17. Hasil dari kode program berikut adalah... #include <iostream>using namespace std; int main() { int x = 157, y = ⁸; int hasil; hasil = x & y; cout<<hasil; return 0;} O 1 O 5 O 2 O 4 O 3 ​


Jawaban:

Kode program tersebut akan menghasilkan output berupa angka desimal, yaitu 8.

Penjelasan:

Baris pertama mendeklarasikan variabel x dengan nilai 157 dan variabel y dengan nilai ⁸. Namun, terdapat kesalahan penulisan pada variabel y yaitu menggunakan karakter superskrip yang salah, sehingga programnya akan mengalami error saat dijalankan.

Namun, bila kita asumsikan karakter superskrip itu sebenarnya adalah angka 8, maka program akan berjalan dan menghasilkan output 8.

Pada baris selanjutnya, nilai variabel hasil dihitung dengan operasi bitwise AND antara nilai variabel x dan y (dalam biner). Nilai biner 157 adalah 10011101, sedangkan nilai biner 8 adalah 00001000. Jika kedua nilai tersebut di-operasikan dengan AND, maka hasilnya adalah 00001000 (dalam biner) atau 8 (dalam desimal).

23. Jelaskan fungsi dari “using name space std” !


Using namespace std adalah salah satu instruksi yang dapat kita berikan kepada aplikasi compiler terkait penggunaan fungsi dalam sebuah kerangka yagn sama.

Pembahasan

Pada kesempatan ini, soal memitna kit auntuk menjelaskan fungsi tersebut. Berikut kakak akan mencoba menjelaskannya secara sederhana.

USING NAMESPACE STD MERUPAKAN PERINTAH YANG DAPAT KITA GUNAKAN UTNUK MENYAJIKAN PERINTAH ATAU DEKLARASI KEPADA SEBUAH APLIKASI COMPILER YANG MENYATAKAN BAHWA KITA AKAN MENGGUNAKAN SELURUH BERKAS, CLASS, ATAU FUNGSI YANG MENJADI BAGIAN DARI SEBUAH NAMESPACE STD YANG BERSANGKUTAN.

BERKAT PENGGUNAAN FUNGSI INI, KITA DAPAT MENGHINDARI KESALAHAN BERUPA TIDAK DIKENALINYA COUT DAN CIN OLEH SISTEM AKIBAT KETIADAAN SYNTAX STD SEKALIGUS SEBAGAI PENGENAL YANG UNIK DARI CIN DAN COUT YANG INGIN KITA GUNAKAN DARI SELURUH KOMBINASI MAUPUN INSTRUKSI YANG TERSEDIA.
