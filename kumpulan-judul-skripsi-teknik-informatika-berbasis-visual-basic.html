Karena Visual Basic di develop oleh Microsoft pada tahun 1991 yang dipakai oleh Windows untuk membangun operating system mereka.

PEMBAHASAN

Visual Basic merupakan sebuah bahasa tingkat tinggi. Basic disini merupakan singkatan dari "Beginners All-purpose Symbolic Instruction Code" yang berarti kode instruksi komputer bagi pemula yang bisa digunakan untuk berbagai hal (all-purpose).

Sebenarnya Microsoft mengeluarkan Visual Basic for Mac pada tahun 2019 lalu. Kita bisa mengembangkan aplikasi dan permainan untuk iOS, Android, dan Web menggunakan .NET di Visual Basic for Mac ini.

5. Salah satu contoh pemrograman visualyang berorientasi objek berbasis Windowsadalah ....a. Visual Basicb. GambasPascald. Git HubC.​


Salah satu pemrograman visual yang berorientasi objek dan berbasis Windows adalah a. Visual Basic (VB). Visual Basic adalah bahasa pemrograman yang menggunakan antarmuka grafis untuk membuat aplikasi Windows. Bahasa ini menggabungkan konsep pemrograman objek dengan kemudahan penggunaan untuk mengembangkan aplikasi desktop yang interaktif. Visual Basic juga mendukung pengembangan aplikasi web dengan menggunakan teknologi .NET.

Pembahasan

Pemrograman visual yang berorientasi objek berbasis Windows merujuk pada pendekatan pengembangan perangkat lunak yang menggunakan bahasa pemrograman visual dan berfokus pada konsep pemrograman berorientasi objek. Dalam konteks Windows, ini berarti menggunakan bahasa pemrograman dan lingkungan pengembangan yang mendukung pembuatan aplikasi yang kompatibel dengan sistem operasi Windows.

Berikut adalah beberapa bahasa pemrograman visual yang berorientasi objek dan berbasis Windows:

Visual Basic .NET (VB.NET) adalah bahasa pemrograman yang dikembangkan oleh Microsoft. C#: C# (C sharp) adalah bahasa pemrograman modern yang dikembangkan oleh Microsoft. Visual C++ adalah versi yang ditingkatkan dari bahasa pemrograman C++. Delphi adalah lingkungan pengembangan perangkat lunak yang menggunakan bahasa pemrograman Object Pascal. PowerBuilder adalah lingkungan pengembangan aplikasi yang dirancang untuk membuat aplikasi bisnis berbasis Windows. Lazarus adalah lingkungan pengembangan perangkat lunak bebas dan sumber terbuka yang memungkinkan pengembangan aplikasi dengan menggunakan bahasa pemrograman Object Pascal.

22. apa yg di maksud dengan visual studio dan visual basic?​


Jawaban:

Visual Studio merupakan sebuah perangkat lunak lengkap (suite) yang dapat digunakan untuk melakukan pengembangan aplikasi, baik itu aplikasi bisnis, aplikasi personal, ataupun komponen aplikasinya, dalam bentuk aplikasi console, aplikasi Windows, ataupun aplikasi Web.

Visual Studio mencakup kompiler, SDK, Integrated Development Environment (IDE), dan dokumentasi.

Visual Studio dapat digunakan untuk mengembangkan aplikasi dalam native code (dalam bentuk bahasa mesin yang berjalan di atas Windows) ataupun managed code.

Visual Basic (VB) merupakan sebuah bahasa pemrograman yang menawarkan Integrated Development Environment (real) visual untuk membuat program perangkat lunak berbasis sistem operasi Microsoft Windows dengan menggunakan model pemrograman (COM).

Visual Basic merupakan turunan bahasa pemrograman BASIC dan menawarkan pengembangan perangkat lunak komputer berbasis grafik dengan cepat.

Kata Kunci: Visual Studio, Visual Basic
