2. Soal menyangkut tntang *Transformator*#Tolong di jawab bsk dikumpul


diketahui :

Ns = 5.000 lilitan

Ip = 2 A

Is = 1 A

Vs = 3.000 v

ditanya :

a)  Np

b)  Vp

jawab :

a)  Np / Ns  =  Is / Ip

   Np / 5.000  =  1 / 2

   Np  =  5.000 x 1/2

          =  2.500 lilitan


b)  Vp / Vs  =  Is / Ip

    Vp / 3.000  =  1/2

     Vp  =  3.000 x 1/2

            =  1.500 v
