14. tolong di jawab soal mat-nya 2 jam lagi pr-ku dikumpulkan.​


Jawaban:

1.) C. 3.375 cm³

2.) B. 4.576 cm²

3.) D. 3.080 cm³

Penjelasan dengan langkah-langkah:

1. V = s x s x s x 1 cm³

= 15 x 15 x 15 x 1 cm³

= 3.375 cm³

2. L = (2xpxl) + (2xpxt) + (2xlxt) x 1 cm²

= (2x28x20) + (2x28x36) + (2x20x36) x 1 cm²

= 1.120 + 2.016 + 1.440 x 1 cm²

= 4.576 cm²

3. r = d : 2

= 14 cm : 2

= 7 cm

V = π x r x r x t x 1 cm³

= 22/7 x 7 x 7 x 20 x 1 cm³

= 3.080 cm³

p = 28 cm.

ℓ = 20 cm.

Jawaban:

Luas permukaan balok

• Luas selimut 264 cm²

• Jari-jari alas (r) 6 cm

• Garis pelukis

• Tinggi kerucut

[tex] = \pi {r} {s}[/tex]

[tex]t = \sqrt{{14} ^{2} - {6}^{2}} [/tex]

[tex]t = \sqrt{{196} - 36}[/tex]

[tex]t = \sqrt{160} [/tex]

t = 12,65 cm
