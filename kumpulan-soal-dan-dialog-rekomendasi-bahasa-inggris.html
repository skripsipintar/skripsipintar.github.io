2. dialog rekomendasi bahasa Inggris dalam 1 halaman​


DIALOG TENTANG LIBURAN

Vian: Hi, Bima! Are you going to go home?

Bima: Yes. I have to pack because tomorrow I’m going to Bali.

Vian: Oh, yeah, tomorrow is a holiday.

Bima: That’s right.

Vian: Will you go there with your parents?

Bima: No. My parents still have a lot of work. I’ll go with my brother and my uncle.

Vian: Oh, I see. Bali is nice, there are many beaches there. Have fun there!

Bima: Thank you. How about you? Where are you going to go?

Vian: Oh, maybe I’ll go to my grandparents’ house. I haven’t been there in a long time.

Bima: That’s nice. Please send my regards to them.

Vian: Okay. See you, Bim!

Bima: See you!

DIALOG TENTANG DISEKOLAH

Mrs. Ayu: Okay, Students. I think we have learned many things yesterday. Today I want to you to have fun.

Alfina: Hooray! Does it mean that we can play outside, Ma’am?

Mrs. Ayu: No, it doesn’t. What I mean is we will have a game indoor. But, the game is about the last material. So, we will learn as well as play. Do you guys want to do this?

Students: Yes!

Mrs. Ayu: Okay. First, what is our last material?

Kean: It’s multiples of numbers, Ma’am.

Mrs. Ayu: That’s right. Now, we will play the game of multiples of numbers. Every one of you need to mention the multiples of a number that I’m going to mention. But, I’ll give the rules. I will mention a number that shouldn’t be mentioned. For example, the multiples of 3. But you should not mention the number with 2 in it. So, if there is a number like 21, you can change it by clapping instead. Do you understand?

Tara: Then what if we forget and still mention it, Ma’am?

Mrs. Ayu: The penalty is you have to quit the game. Someone who stay at the end of the game will be the winner. Okay?

Students: Okay!

B: No, I haven't. Is it any good?

Dialog 2:

A: I'm thinking of watching a movie tonight. Any recommendations?

B: Well, if you're into action, I heard the new superhero movie is really entertaining.

A: Sounds like a plan. Let's get tickets for that one.

B: Have you read the latest bestseller in the mystery genre? It's been getting rave reviews.

A: Thanks, I'll look it up at the bookstore.

B: If you're looking for relaxation, a beach destination like Bali would be perfect.

A: Bali it is then. Thanks for the recommendation!

B: How about trying that new recipe for chicken curry? It's simple and delicious.

A: That sounds good. I'll give it a shot.

B: Well, based on the features and reviews, the latest model from XYZ seems to be a hit.

A: I'll check it out. Thanks for the advice!

A: Photography sounds interesting. I'll look into it.

5. Tuliskan dialog tentang rekomendasi​


Jawaban:

Juan: Hai Farel, apakah Anda punya waktu luang hari Minggu ini?

Farel: Baiklah, biar saya ingat. Saya tidak berpikir saya memiliki acara hari Minggu ini. Apa Juan?

Juan: Iya sebenarnya saya ingin mengajak Anda untuk pergi ke pantai.

Farel: Maaf Juan. Saya pikir itu ide yang buruk. Suatu saat cuaca sangat panas, pasti di pantai juga akan sangat panas.

Juan: Saya setuju dengan Anda. Apakah Anda punya saran tentang tempat lain?

Farel: Ya. Bagaimana kalau kita bermain game di rumahku? Saya punya beberapa judul game baru.

Juan: Ide bagus. Game apa itu?

Farel: Sebuah game horor berjudul "Home Sweet Home".

Juan: Maafkan saya Farel. Saya kurang suka game dengan genre horor.

Farel: Oke Juan. Menurutmu game apa yang harus kita mainkan nanti?

Juan: Bagaimana dengan game petualangan?

Farel: Ide bagus. Saya juga memiliki game bertema petualangan baru, dan saya menyukainya.

Juan: Baiklah, Farel, saya pikir lebih baik kita memainkan game petualangan karena kita berdua menyukainya.

Farel: Tentu saja Juan. Sampai jumpa di hari Minggu.

Juan: Sampai jumpa Farel.
