Menurut saya, beberapa perintah dasar untuk mengerjakan soal diatas adalah:

1. ls
untuk menampilkan isi sebuah direktori

2. cd [nama direktori]
untuk masuk ke direktori
contoh: cd /home/ezra/video

3. mkdir [nama direktori baru]
untuk membuat direktori baru
contoh: mkdir ezra

4. cp [nama file / direktori] [direktori tujuan]
untuk meng-copy / menyalin file / direktori ke direktori tujuan
contoh: cp ezra.txt /home/ezra/video

5. mv [nama file / direktori] [direktori tujuan]
untuk meng-cut / memindah file / direktori ke direktori tujuan

6. touch [nama file]
untuk membuat file kosong
contoh: touch ezra.txt

7. nano [nama file]
untuk mengedit file teks
contoh: nano ezra.txt
*jika telah selesai mengedit, save dengan cara tekan Ctrl + X, ketik "y", lalu Enter

8. clear
untuk membersihkan semua teks yang ada di layar monitor

9. adduser [nama user]
untuk membuat user pada linux

10. cal
menampilkan kalender

11. time
menampilkan waktu dan tanggal sistem

12. whoami
menampilkan informasi user yang sedang aktif

*Penjelasan tambahan: untuk masuk dengan akun root, saat login, gunakan username "root" dengan password akun yang pertama kali dibuat saat instalasi linux.

8. Buatlah soal+jawaban Analogi jaringan komputer


Jawaban:

Pada saat ini, istilah komputer awalnya dianggap sebagai seperangkat alat yang terdiri atas monitor, keyboard, Mouse, hingga Personal Computer (PC) desktop yang relatif berat dan sulit dibawa telah berevolusi menjadi lebih ramping dan berteknologi micro. Dengan bentuk yang lebih ringan dan kecil, teknologi komputer yang terdapat pada peranti tersebut memiliki kecepatan dan kinerja yang tak kalah dari komputer atau PC, seperti PC tablet, Ponsel pintar, ponsel genggam, netbook, notebook, dan sebagainya. Mengapa teknologi tersebut bisa dikategorikan sebagai komputer? karena peranti-peranti tersebut memiliki alat input seperti keyboard, alat pemrosesan seperti processor, RAM, hard disk, dan alat output berupa layar monitor dengan ukuran tertentu. Dengan peranti tersebut, dapat saling berhubungan satu sama lain dalam sebuah jaringan. Namun, ketika mesin tersebut Hanya bekerja sendiri dan tidak terhubung dengan komputer lain, disebut sebagai komputer stand alone.
