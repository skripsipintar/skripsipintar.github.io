12. jelaskan mengenai format penyimpanan doc dan xls​


doc adalah sebuah extensi format file, yang disimpan dengan menggunakan versi microsoft word 2003 dan versi microsoft word yang lebih lama, bisa dikatakan format ini sudah ada sejak Word Perfect diluncurkan. format ini sudah di hak patenkan oleh microsoft, agar pembuat software ini tidak dapat membuat software lain, yang dapat menghasilkan format doc tersebut, sehingga pada saat itu banyak software tidak dapat menganalisa atau membaca format tersebut dengan baik

xls merupakan file penyimpanan default untuk Microsoft Excel dengan versi 2003 kebawah. dalam file ini bisa menyimpan format Macro VBA sehingga cukup sering digunakan. ukuran file pada format ini biasanya jauh lebih besar jika dibandingkan dengan format default Excel 2007 keatas.

Jawaban:

Format .doc merupakan format penyimpanan dari Microsoft Office Word versi lama sampai ke Microsoft Office Word 2003, sedangkan untuk format .docx merupakan format penyimpanan dari Microsoft Office Word 2007 dan 2010. Secara umum, format .docx tidak bisa dibaca oleh Microsoft Office Word versi lama dan 2003.

35. Apa yang anda akan lakukan kedepan jika sudah lulus dari smk jurusan otomotif


Jawaban:

1. Melanjutkan ke Pendidikan Tinggi: Setelah lulus dari SMK, Anda dapat mempertimbangkan untuk melanjutkan pendidikan ke jenjang perguruan tinggi atau politeknik yang menyediakan program studi terkait otomotif. Gelar sarjana atau diploma lebih tinggi dapat membuka kesempatan kerja dan karier yang lebih luas di bidang otomotif.

2. Magang atau Praktek Kerja: Anda bisa mencari kesempatan untuk magang atau melakukan praktek kerja di bengkel atau perusahaan otomotif. Ini akan membantu Anda mendapatkan pengalaman praktis dan memperluas pengetahuan dan keterampilan dalam dunia otomotif.

3. Mencari Pekerjaan: Anda dapat mencari pekerjaan sebagai mekanik, teknisi otomotif, atau berbagai peran lainnya di industri otomotif. Pastikan untuk melamar pekerjaan yang sesuai dengan keahlian dan minat Anda.

4. Mengikuti Sertifikasi: Pertimbangkan untuk mengikuti program sertifikasi resmi di bidang otomotif. Sertifikasi dapat meningkatkan kredibilitas Anda sebagai profesional otomotif dan membuka kesempatan lebih banyak dalam industri ini.

5. Mengembangkan Keterampilan Tambahan: Selain keahlian teknis, Anda juga dapat mengembangkan keterampilan tambahan seperti kemampuan berkomunikasi, kepemimpinan, dan manajemen proyek. Keterampilan ini akan membantu Anda menjadi lebih kompetitif di pasar kerja.

6. Membuka Usaha Sendiri: Jika Anda memiliki jiwa wirausaha, Anda dapat mempertimbangkan untuk membuka bengkel atau usaha lain yang berhubungan dengan dunia otomotif. Ini membutuhkan persiapan yang matang dan pengetahuan tentang manajemen bisnis.

7. Tetap Belajar: Industri otomotif terus berkembang dan berubah dengan cepat. Selalu berusaha untuk tetap belajar dan mengikuti perkembangan terbaru dalam teknologi otomotif dan tren industri.

Jawaban:

sebagai lulusan SMK jurusan otomotif, saya memiliki berbagai pilihan untuk langkah selanjutnya. Beberapa pilihan yang dapat dipertimbangkan setelah lulus dari SMK jurusan otomotif antara lain:

1. Perguruan Tinggi: saya dapat melanjutkan pendidikan ke perguruan tinggi untuk mengambil program studi yang terkait dengan otomotif, seperti Teknik Otomotif atau Teknik Mesin.

2. Magang atau Kerja: saya dapat mencari kesempatan magang atau bekerja di bengkel otomotif, perusahaan manufaktur otomotif, atau dealer mobil untuk memperluas pengalaman dan keterampilan praktis saya.

3. Wirausaha: Jika saya memiliki minat dalam berwirausaha, saya dapat membuka bengkel otomotif sendiri atau menjalankan bisnis terkait lainnya di bidang otomotif.

4. Sertifikasi Profesional: saya dapat memperoleh sertifikasi profesional di bidang otomotif, seperti Automotive Service Excellence (ASE) certification, yang dapat meningkatkan kredibilitas dan peluang kerja saya

5. Pendidikan Lanjutan: saya juga dapat mengikuti pelatihan atau kursus tambahan untuk mengasah keterampilan tertentu, seperti teknologi otomotif terbaru atau teknik khusus dalam perbaikan mobil.
