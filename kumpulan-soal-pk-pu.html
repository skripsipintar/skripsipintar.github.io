6. 10. PK yang akar-akarnya kurang 3 dari akar-akar pk x² + 5x - 7 = 0 adalah....Tolong bantu jawab yah kakak", soalnya besok mau dikumpulkan​


Jawab:

﻿x² + 11x + 17 = 0

Penjelasan dengan langkah-langkah:

﻿ax² + bx + c = 0

﻿x² + 5x - 7 = 0

x﻿₁ + x₂ = -b/a = -5/1 = -5

﻿x₁x₂ = c/a = -7/1 = -7

﻿α = x₁ - 3

﻿β = x₂ - 3

﻿x² - (α + β)x + αβ = 0

﻿x² - (x₁ - 3 + x₂ - 3)x + (x₁ - 3)(x₂ - 3) = 0

﻿x² - (x₁ + x₂ - 6)x + x₁x₂ - 3(x₁ + x₂) + 9 = 0

﻿x² - (-5 - 6)x - 7 - 3(-5) + 9 = 0

﻿x² + 11x + 17 = 0

﻿Cara cepat

﻿Persamaan kuadrat ax² + bx + c = 0 akar-akarnya x₁ atau x₂. Persamaan kuadrat yang akar-akarnya x₁ - m atau x₂ - m adalah a(x + m)² + b(x + m) + c = 0

﻿1(x + 3)² + 5(x + 3) - 7 = 0

﻿x² + 6x + 9 + 5x + 15 - 7 = 0

﻿x² + 11x + 17 = 0

Mr KNO3 = 39+14+48 = 101 gr/mol
m = 5,05 gr
mol KNO3 = m/Mr = 5,05 gram/101 gr/mol = 0,05 mol

STP, 1 mol = 22,4 L, maka
V KNO3 (STP) = n x 22,4 = 0,05 x 22,4 =1,12 Liter
