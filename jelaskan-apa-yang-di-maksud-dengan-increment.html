1. Apa yang dimaksud dengan pre-increment dan post increment


Pre-increment adalah penambahan 1 angka dari konstanta terkait sebelum proses selanjutnya dilakukan.

Post-increment adalah penambahan 1 angka dari konstanta terkait sesudah segala proses dilakukan


2. jelaskan tentang operator increment dan decrement


Intcrement digunakan untuk pengulangan yang nilainya bertambah/naik ke atas.
Penulisan: a++, a+=1, x+=5

Sedangkan decrement digunakan untuk pengulangan yang nilainya semakin berkurang/turun ke bawah.
Penulisan: x--, a-=1, x-=10

Penjelasan:

Increment artinya penambahan angka 1.

Pada soal di atas, perhitungan increment menjadi 10011011 + 1

Untuk lebih mudah melakukan penjumlahan bilangan biner, susunannya vertikal. Amati di lampiran:

- pada digit paling kanan (ditunjukkan tanda panah merah) penjumlahan 1+1 hasilnya 10 namun ditulis 0 sebab angka 1 akan dijumlahkan di digit berikutnya

- pada digit kedua dari kanan (ditunjukkan tanda panah biru) angka 1 dijumlahkan dg angka 1 hasil perhitungan digit sebelumnya. Hasilnya 10 namun ditulis 0 sebab angka 1 akan dijumlahkan di digit berikutnya.

- pada digit ketiga dari kanan (ditunjukkan tanda panah hijau) angka 0 dijumlahkan dg angka 1 hasil perhitungan digit sebelumnya & hasilnya 1


25. increment bilangan biner 1111001


Jawaban:

Untuk menambahkan 1 pada bilangan biner 1111001, kita bisa mengikuti tahapan berikut:

- Tambahkan 1 pada bit terakhir (dari kanan), sehingga menjadi 1111010.

- Jika hasilnya melebihi batas maksimum dari setiap bit (yaitu 1), maka bawa ke bit sebelah kiri dan tambahkan 1 pada bit tersebut.

- Ulangi langkah 2 hingga tidak ada bit yang melebihi batas maksimum.

Dengan demikian, hasil increment dari bilangan biner 1111001 adalah 1111010.

38. Buatlah algoritma untuk menampilkan tabel konversi dari centimeter ke inci dan dari meter ke feet dan yard mulai dari nilai 0 sampai 100 dengan increment 5


Buatlah algoritma untuk menampilkan tabel konversi dari centimeter ke inci dan dari meter ke feet dan yard mulai dari nilai 0 sampai 100 dengan increment 5

Untuk menjawab soal diatas, pertama-tama kita perlu mengetahui konversi satuan tersebut.

1 centimeter = 0,3937 inch.

1 meter = 3,28 feet  = 1,093 yard.

Algoritma akan saya tuliskan dalam bahasa natural dan kode C++

1. algoritma untuk menampilkan centimeter ke inchi

Sebut saja nilai 0 sampai 100 adalah n. Maka selama n masih kurang dari sama dengan 100 => cetak { n centimeter  = n x 0,3937 inchi }, n + 5

2. algoritma untuk menampilkan meter ke feet dan yard

Sebut saja nilai 0 sampai 100 adalah n. Maka selama n masih kurang dari sama dengan 100 => cetak { n meter = n x 3,28 feet = n x 1,093 yard}, n + 5

Karena konversi membutuhkan operasi dengan perkalian angka yang memiliki koma (tidak bulat), maka variabel n harus bertipe float.

Dalam bahasa pemrograman akan menjadi seperti berikut ini

int main()

{

   for(int i = 0; i <= 100; i+=5)

   {

       float n = i;

       printf("%f centimeter = %f inch\n", n, n * 0.3937F);

   }

   for(int i = 0; i <= 100; i+=5)

   {

       float n = i;

       printf("%f meter = %f feet = %f yard\n", n, n * 3.28F, n * 1.093 F);

   }

   getch();

   return(0);

}

Penjelasan:

Dalam program, kita menggunakan perulangan dengan "for". for disini membutuhkan variabel integer, namun pada perhitungan kita membutuhkan variabel float. Sehingga kita buat variabel float n yang nilainya adalah int i dari dalam "for" tersebut.

Untuk mencetaknya, karena variabel bertipe float, maka pada fungsi printf kita tuliskan vairabel dengan %f.
