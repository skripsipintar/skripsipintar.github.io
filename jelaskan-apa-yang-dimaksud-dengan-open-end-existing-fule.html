3. Apa yang disebut dengan open dan existing file?


Kelas           : XII (3 SMA)

Pelajaran     : TI

Kategori      : Dasar Dasar Microsoft Access

Kata Kunci  : Open Existing File, Database, File

 

 

 

Microsoft Access atau lengkapnya Microsoft Office Access merupakan aplikasi berbasis komputer yang menawarkan beberapa fungsi dalam mengelola data dalam kehidupan sehari-hari maupun untuk kegiatan perkantoran.

 

Dalam menjalankan fungsi pada program Microsoft Access terdapat banyak jenis perintah, salah satunya adalah OPEN END EXISTING FILE. 


Jenis perintah yang ini dijalankan jika pengguna hendak mengakses atau membuka file di database yang sebelumnya telah dibuat. Artinya file yang hendak dibuka tersebut telah exist atau ada dan hendak dibuka kembali.

7. Jelaskan apa yang dimaksud dengan open market polic


Operasi Pasar Terbuka (Open Market Operation)
Operasi pasar terbuka adalah cara mengendalikan uang yang beredar dengan menjual atau membeli surat berharga pemerintah (government securities).
Jika ingin menambah jumlah uang beredar, pemerintah akan membeli surat berharga pemerintah. Namun, bila ingin jumlah uang yang beredar berkurang, maka pemerintah akan menjual surat berharga pemerintah kepada masyarakat. Surat berharga pemerintah antara lain diantaranya adalah SBI atau singkatan dari Sertifikat Bank Indonesia dan SBPU atau singkatan atas Surat Berharga Pasar Uang.

24. sebutkan fungsi dari blank database,blank data access page,project existing data,open and existing file


Sebutkan fungsi dari blank database,blank data access page,project existing data,open and existing file, istilah-istilah tersebut adalah istilah yang terdapat pada saat kita akan menggunakan microsoft acces. Seblumnya mari kita bahas terlebih dahulu secara singkat apa itu microsoft access,microsoft access adalah suatu aplikasi buatan microsoft yang berfungsi untuk mengelola database sederhana. Kemudahan penggunaan microsoft access membuat banyak orang menggunakan aplikasi microsoft access.

Pada penggunaan microsoft access, terdapat beberapa pilihan, beberapa pilihan tersebut adalah

1. Blank database, pilihan ini merupakan pilihan yang bisa memberikan kita kebebasan dalam membuat database pada microsoft access, karena pada pilihan blank database, database yang kita buat benar-benar kosong kita dapat menambahkan tabel, form dan lainnya.

2. Blank data access page, pada pilihan ini kita akan dibantu oleh microsoft access dalam pembuatan database, karena database yang kita buat akan dilakukan secara bertahap.

3. Project existing data, pada pilihan ini kita akan menggunakan template database yang sudah ada pada microsoft access.

4. Open and existing file, pada pilihan ini kita akan menggunakan database yang sudah pernah kita buat sebelumnya.

Demikian penjelasan mengenai istilah yang ada di microsoft access, istilah tersebut muncul pada tampilan awal microsoft access. Untuk membaca materi lainnya mengenai microsoft access dapat dibaca pada link berikut
