12. Sebuah program yang digunakan sebagai alat bantu untuk mencari informasi di internet disebut…

c. Search engine

Pembahasan:

Search engine misalnya adalah Google, Bing atau Yahoo, yang membantu mencari informasi seperti situs yang hendak dituju atau berita terbaru.

13. E-mail adalah salah satu aplikasi internet yang berfungsi untuk…

b. Surat-menyurat    

Pembahasan:

E-mail adalah singkatan dari “electronic mail” atau “surat elektronik”, dn digunakan untuk mengirim pesan surat melalui jaringan internet. Contoh penyedia layanan e-mail misalnya adalah Gmail, Hotmal dan ProtonMail.

14. Proses pengambilan file dari sebuah computer server di internet ke computer kita disebut…      

a. Download    

Pembahasan:

Saat kita melakukan download, maka file yang kita minyta akan dikirim dari server melalui jaringan internet, menuju komputer kita. Setelah selesai, file ini dapat dibuka atau dijalankan (bila file aplikasi).

15. ISP (Internet Service Provider) adalah……

b. Perusahaan penyedia jasa layanan koneksi internet

Pembahasan:

Untuk mengakses jaringan internet, maka komputer kita harus terhubung melalui perusahaan yang menyediakan jasa koneksi internet ini, baik secara nirkabel (wireless) seperti melalui telepon genggam, atau melalui sambungan kabel fiber optic.

16. Kita dapat menulis dan melampirkan gambar atau foto dalam blog atau facebook kita. Dalam hal ini kita dapat melakukan….

b. Upload    

Pembahasan:

Upload adalah kebalikan dari download, di mana kita dalam upload mengirim file dari komputer melalui jaringan internet ke server.

17. Seseorang yang melakukan akses internet disebut…..

b. Netter      

Pembahasan:

Netter adalah sebutan untuk pengguna jaringan internet, sementara user lebih ke pengguna suatu komputer atau situs tertentu

18. Program yang ditempatkan di internet dan dibagikan secara gratis adalah….

a. Freeware    

Pembahasan:

Program freeware dapat digunakan tanpa harus membayar lisensi dan dapat digandakan tanpa batasan. Contoh freeware yang banyak digunakan misalnya adalah browser Firefox untuk mengakses halaman web.

Komputer dapat terhubung satu sama lain (jaringan peer-to-peer) atau dihubungkan dengan bantuan sebuah server.
