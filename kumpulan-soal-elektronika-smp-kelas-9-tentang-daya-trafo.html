1. Sebuah transformator mempunyai kumparan primer dan sekunder dengan jumlah lilitan masing-masing 500 dan 5000, dihubungkan dengan jaringan bertegangan arus bolak-balik 220 V. Berapakah tegangan keluarannya?  

Pembahasan

Diketahui :

Kumparan primer (NP) = 500 lilitan  

Kumparan sekunder (Ns) = 5000 lilitan  

Tegangan primer (VP) = 220 Volt

Ditanya : Tegangan sekunder (VS)

Jawab  :

Rumus transformator :

Vs / Ns = Vp / Np

Vs / 5000 = 220 / 500

Vs / 5000 = 0,44

Vs = (0,44)(5000)  

Vs = 2.200 Volt

7. Efisiensi sebuah trafo 75% jika daya yg keluar dari trafo525 watt maka daya yg masuk kedalam trafo adalah


"Efesiensi Transformator 75% jika daya yang keluar 525 watt berapa daya yang masuk  ?"


Penjelasan Dengan langkah-langkah :

Dari hubungan jumlah lilitan, tegangan, dan arus secara matematis di Rumuskan sebagai berikut  

[tex]\frac{VP}{VS}[/tex] = [tex]\frac{NP}{NS}[/tex] = [tex]\frac{IS}{IP}[/tex]

Untuk efisiensi Transformator di Rumuskan sebagai berikut :  

μ = [tex]\frac{PS}{PP}[/tex] x 100%  μ = [tex]\frac{VS.IS}{VP.IP}[/tex] x 100%  μ = [tex]\frac{NS.IS}{NP.IP}[/tex] x 100%  

Keterangan  

VP = Tegangan Primer (volt)

VS = Tegangan Sekunder (volt)

NP = Lilitan Primer  

NS = Lilitan Sekunder

IS = Arus Sekunder ( A )

IP = Arus Primer (A)

PS = Tegangan Sekunder (watt)

PP = Tegangan Primer (watt)

μ = Efisiensi (%)  

=================================================================

Pembahasan soal  


Diketahui :  

μ   =  Efesiensi  =  75%

PS =  Daya sekunder   =  525 watt


Ditanyakan :

PP  =  Daya Primer  = ...   ?


Penyelesaian :

Gunakan Rumus :

%μ  =  PS/PP x 100%

75%  =  525/PP x 100%

75PP = 52.500

   PP = 52.500/75

   PP = 700


Jadi Daya Masukkan sebesar 700 watt

Berikut penjelasan dari jawaban diatas :





Diketahui :

Ps = 63 watt

η = 90%



Ditanyakan : Pp..........?





Dijawab :

Trafo merupakan alat yang digunakan untuk mengubah tegangan listrik bolak balik menjadi lebih besar atau lebih kecil.

Trafo sendiri dibagi menjadi 2 yaitu trafo step up (menaikan tegangan) dan trafo step down (menurunkan tegangan). Adapun ciri ciri trafo step up yaitu nilai Vs > Vp (tegangan sekunder lebih besar dibandingkan tegangan primer) , Ns > Np (jumlah lilitan sekunder lebih besar dibandingkan jumlah lilitan primer), Is < Ip (kuat arus sekunder lebih kecil dibandingkan dengan kuat arus primer). Untuk ciri ciri trafo step down yaitu nilai Vs < Vp (tegangan sekunder lebih kecil dibandingkan tegangan primer) , Ns < Np (jumlah lilitan sekunder lebih kecil dibandingkan jumlah lilitan primer), Is > Ip (kuat arus sekunder lebih besar dibandingkan dengan kuat arus primer).





Untuk mencari rumus efisiensi transformator adalah sebagai berikut :

η = (Is x Vs / Ip x Vp) x 100% atau η = (Ps/Pp) x 100%

Ket:

η = efisiensi trafo

Pp = daya masukan primer

Ps = daya keluaran sekunder

Vp = Tegangan pada kumparan primer (masukkan)

Vs = Tegangan pada kumparan sekunder (keluaran)

Ip = Arus pada kumparan primer

Is = Arus pada kumparan sekunder



Untuk mencari daya sekundernya rumisnya sebagai berikut :

η = (Ps/Pp) x 100%

90% = (63 / Pp) x 100%

90 x Pp =63 x 100

Pp = 63 x100 /90

Pp = 6300/90

Pp = 70 watt

Jadi besarnya daya primernya adalah 70 watt
