16. Sebuah kisi memiliki 12500 garis per cm. Seberkas sinar monokromatis fatang tegak lurus pada kisi. Bila spectrum orde pertama membuat sudut 30 derajat dengan garis normal pada kisi, maka panjang gelombang sinar tersebut adalah ...


Kategori: Fisika 
Materi: Cahaya 
Kelas: XII SMA IPA 
Kata kunci: Difraksi 

Diketahui :

N = 12500 garis / cm
n = 1
θ = 30º

Ditanya :
λ = __ ?


Pembahasan :

d = 1/N
d = 1/12500 ×10⁻² = 8 × 10⁻⁷ m

d sin θ =  n λ
        λ = d sin θ /  n
        λ = (8 × 10⁻⁷ m) (0,5) /  1
        λ = 4 × 10⁻⁷ m
           = 400 nm
           = 4000 Â

Jadi, panjang gelombang cahaya yang digunakan 4000 Amstrong.

•>. Gunakan rumus umum

~> d sin θ = n λ

P = n.λ.L/d

~> Di P akan terjadi garis gelap jika :

d sin θ = (2n+1).1/2.λ

P = (2n + 1).λ.L/d

dengan ;

d = lebar celah kisi (m)

θ = sudut difraksi (derajat)

λ = panjang gelombang cahaya (m)

n = orde difraksi

p = jarak garis gelap/terang ke terang pusat (m)

L = jarak layar ke kisi (m)

•>. Diketahui ;

d = 1/N.

θ = 30°.

n = 1

Å = 10'-10 m.

Jwab ;

dari rumus diatas kita dapat menggunakan persamaan yg ini :

d.sin θ = n. λ

1/6250.1/2 = λ

0, 5/6250 × 10/10 = λ

5/62500 = λ

1/12500 = λ

λ = 8 × 10'-7 m

λ = 800 nm

= 8000 ångström ✓


31. Seberkas sinar monokromatis dengan panjang gelombang 5×10 -7 m datang tegak lurus pada kisi. Jika spektrum orde kedua membuat sudut 30° dengan garis normal pada kisi,jumlah garis per cm kisi adalah


Jawaban: N = 5000 cm

Penjelasan:

rumus : dsinΘ = mλ

diketahui: λ = 5x10^-7   sudut Θ = 30*

m(orde)= 2

Ditanya: N=?

Jawab: dsinΘ = mλ

d=1/N

1/N .sinΘ = mλ

sinΘ/mλ =N

N= sin 30* / 2 . 5x10^-7 m

N= sin 30* / 2 . 5x10^-5 cm

N= 0,5 / 10x10^-5

N= 0,05x10^5 cm

N= 5000 cm
