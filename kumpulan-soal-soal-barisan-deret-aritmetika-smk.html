5. SOAL DERET ARITMETIKA Tentukan rumus jumlah n suku pertama deret aritmetika jika U7=15 dan Sn=168.


 Sn      =  n/2 (U1 + Un)
 S7      =  7/2 (U1 + U7)
168     = 7/2 (U1 + 15)
168x2 = 7U1 + 105
336     = 7U1 + 105
7U1    = 336 - 105
            = 231
  U1     = 231/7 = 33
 
rumus jumlah n suku pertama dari yg diatas adalah :
Sn = n/2 (U1 + Un)
[tex]Sn = (a + Un) . n/2 \\ 168 = (a + 15) . 7/2 \\ 168 = (7a + 105 ) / 2 \\ 168(2) = 7a + 105 \\ 336 = 7a + 105 \\ 231 = 7a \\ a = 231 / 7 \\ a = 33[/tex]

contoh soal :

contoh soal :

9. Soal tentang barisan dan deret. Tentukan suku ke-n barisan aritmetika jika diketahui u7 = 7p + 5q dan u11 = 11p + 91


[tex]U_7=7p+5q[/tex]
[tex]a+6b=7p+5q[/tex]...(1)
[tex]U_{11}=11p+91[/tex]
[tex]a+10b=11p+91[/tex]...(2)
kurangi persamaan (2) dengan persamaan (1), sehingga didapatkan:
[tex]4b=4p-5q+91[/tex]
[tex]b= \frac{4p-5q+91}{4}[/tex]
substitusikan [tex]b= \frac{4p-5q+91}{4}[/tex] ke persamaan (1), sehingga diperoleh:
[tex]a+6(\frac{4p-5q+91}{4})=7p+5q[/tex]
[tex]a+6p- \frac{15}{2}q- \frac{273}{2}=7p+5q[/tex]
[tex]a=7p-6p+5q+\frac{15}{2}q+\frac{273}{2}=p+ \frac{25}{2}q+\frac{273}{2}[/tex]
dengan demikian :
[tex]U_n=p+ \frac{25}{2}q+\frac{273}{2} + (n-1)\frac{4p-5q+91}{4}[/tex]

33. SOAL LATIHAN PERTEMUAN 3 Kerjakanlah soal di bawah ini dengan benar! 1. Diketahui deret aritmetika 7 +13 + 19 + ... Tentukan jumlah dua puluh dua suku pertama deret tersebut! 2. Diketahui barisan aritmetika dengan suku pertama = 6 dan bedanya = 4. Tentukan rumus suku ke-n dari barisan tersebut!​


1.

1. Diketahui :

[tex]U1 = 7 \\ U2 = 13 \\ U3 = 19[/tex]

Ditanya :

[tex]S22 = \: ?[/tex]

Dijawab :

Rumus jumlah n suku pertama deret aritmatika :

[tex]Sn = \frac{n}{2} (2a + (n - 1)b)[/tex]

Nilai a :

[tex]a = U1 \\ a = 7[/tex]

Nilai b :

[tex]b = Un - U(n - 1) \\ b = U2 - U1 \\ b = 13 - 7 \\ b = 6[/tex]

Substitusi nilai a dan b ke dalam rumus jumlah n suku pertama deret aritmatika :

[tex]Sn = \frac{n}{2} (2(7) + (n - 1)6) \\ Sn = \frac{n}{2} (14 + 6n - 6) \\Sn = \frac{n}{2} (6n + 8) \\ Sn = n(3n + 4) \\ Sn = 3 {n}^{2} + 4n[/tex]

Jumlah 22 suku pertama :

[tex]S22 = 3 ({22}^{2}) + 4(22) \\ S22 = 3(484) + 88 \\ S22 = 1452 + 88 \\ S22 = 1540[/tex]

2.

Diketahui :

[tex]U1 = a \\ a = 6 \\ b = 4[/tex]

Ditanya :

[tex]Un = \: ?[/tex]

Dijawab :

Rumus suku ke-n barisan aritmatika :

[tex]Un = a + (n - 1)b[/tex]

Substitusi nilai a dan b ke dalam rumus suku ke-n barisan aritmatika :

[tex]Un = 6 + (n - 1)4 \\ Un = 6 + 4n - 4 \\ Un = 4n + 2[/tex]
