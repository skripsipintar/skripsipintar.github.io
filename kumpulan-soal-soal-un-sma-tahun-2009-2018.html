6. (M0difikasi UN 2009) Skor pada kompetisi matematika adalah 4 untuk setiap jawaban yang benar. Selanjutnya, 0 untuk soal yang tidak di jawab dan -1 untuk setiap jawaban salah. Dari 50 soal yang di berikan, Siti tidak menjawab 6 soal dan menjawab salah 5 soal. Berapa skor yang diperoleh Siti?


Jawab:156

Penjelasan dengan langkah-langkah:

6 soal tidak di jawab = 0

5 soal salah di jawab = -5 (-1x5)

(50-11(6+5))39 soal benar

39x4=156

Jawaban:

Skor yang diperoleh Siti adalah 151

Penjelasan dengan langkah-langkah:

Banyak soal yang dijawab benar :

50 - (6+5) = 50 - 11 = 156

Maka total skor :

B = 4 x 39 soal = 156

S = ( -1 ) x 5 soal = ( -5 )

Tj = 0 x 6 soal = 0

Kalimat matematika :

Total skor

( 4 x 39 ) + ( -1 x 5 ) + ( 0 x 6 ) =

156 + ( -5 ) + 0 = 151

Jadi, skor yang diperoleh Siti adalah 151

17. 12. (Modifikasi UN 2009) Skor pada kompetisi matematika adalah 4 untuk setiap jawaban yang be Selanjutnya, 0 untuk soal yang tidak dijawab dan -1 untuk setiap jawaban salah 50 soal yang diberikan, Siti tidak menjawab 6 soal dan menjawab salah 5 soal skor yang diperoleh Siti? ​


Jawab:

Skor = 151

Penjelasan dengan langkah-langkah:

Diketahui :

Skor 4 untuk Jawaban benar

Skor 0 untuk soal tak dijawab

Skor -1 untuk jawaban salah

Jumlah soal = 50

Tidak dijawab = 6

Dijawab salah = 5

Dijawab benar = 50 - (6 + 5) = 50 - 11 = 39

Pembahasan :

(39 x 4) + (6 x 0) + (5 x (-1)) = 156 + 0 -5

                                           = 151

Jadi, skor siti adalah 151

Diket :

Skor Benar : 4

Skor Salah. : -1

Skor Tidak Diisi : 0

Diatanya : Skor Siti

Pembahasan :

Skor Salah : 5 × ( - 1 ) = -5

Skor T.Diisi : 6 × 0 = 0

Skor Benar : 50-6-5 = 39 × 4 = 156

Skor Total : 156 + ( - 5 ) + 0 : 156 -5 + 0 : 151

Jadi skor Siti adalah 151

36. Diketahui persamaan 5x–6=2x+3.Nilai x+5 adalah....(UN tahun 2009)


Jawaban:

3) 5x - 6 = 2x + 3

5x - 2x = 3 + 6 (Sistem pindah ruas)

3x = 9

x = 3

Jadi, x + 5 = 8

Penjelasan dengan langkah-langkah:

[tex] \texttt{}[/tex]

Cari nilai x

[tex]5x - 6 = 2x + 3 \\ \\ 5x - 2x = 3 + 6 \\ \\ 3x = 9 \\ \\ x = 9 \div 3 \\ \\ x = 3[/tex]

[tex] \texttt{}[/tex]

Hitung nilai x + 5

[tex]x + 5 \\ \\ =3 + 5 \\ \\ = 8[/tex]

Maka, nilai x + 5 adalah 8
