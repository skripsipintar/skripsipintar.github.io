Untuk menyelesaikan soal ini, kita akan menggunakan rumus berikut :

\displaystyle\boxed{\boxed{\bf n(S) = n(A \cup B) + n(A \cup B)^{c}}}

n(S)=n(A∪B)+n(A∪B)

c

dengan,

\displaystyle\boxed{\boxed{\bf n(A \cup B) = n(A) + n(B) - n(A \cap B)}}

n(A∪B)=n(A)+n(B)−n(A∩B)

dimana :

○ n(S) = banyak anggota himpunan semesta

○ n(A) = banyak anggota himpunan A

○ n(B) = banyak anggota himpunan B

○ n(A ⋃ B) = banyak anggota gabungan himpunan A dan himpunan B

○ n(A ⋂ B) = banyak anggota irisan himpunan A dan himpunan B
