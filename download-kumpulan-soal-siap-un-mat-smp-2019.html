U5 = a+4b

3 = a

maka, U20 = a + 19b = 3 + 19(5) = 3 + 95 = 98

diket: U5 = 23

dit: U20?

jawab:

U5 = 23 => a + 4b = 23 => persamaan 1

U7 = 33 => a + 6b = 33 => persamaan 2

eliminasikan persamaan 2 dgn persamaan 1:

=> a + 6b = 33

a + 4b = 23

= 2b = 10

b = 10/2 = 5

substitusikan persamaan 2 untuk mencari a:

a + 6b = 33

a + 6(5) = 33

a = 33 - 30

= 3

U20 = a + 19b

= 3 + 19(5)

= 3 + 95 = 98

jadi, U20 pada barisan aritmatika tsb adalah 98.
