suatu persamaan garis lurus y = x+1. Persamaan garis yangtegak lurus dengan garis tersebut dan melalui titik (2,-1) adalah ......
a. x - y +1 = 0
b. x + y +1 = 0
c. x = y -1
d. x + y -1 = 0

Pembahasan :
Cari gardien garis tersebut, m = 1
m tegak lurus = -1
m ( x-x1) = y - y1
-1 ( x -2 ) = y +1
-x+2 = y+1
x + y -1 = 0
