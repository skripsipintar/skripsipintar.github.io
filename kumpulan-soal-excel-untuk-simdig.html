15. MOD ialah salah 1 fungsi dalam Ms. Excel untuk mencari sisa hasil bagi data angka. Format penulisan fungsi MOD yaitu:

=MOD(angka;dibagi_berapa)

Berdasarkan soal di atas, fungsi MOD digabung dg fungsi IF dan sesuai kaidah penulisan di atas, jawaban yg tepat adalah C

Pembacaan fungsinya menjadi:

"Jika angka dalam sel C2 dibagi 2 sama dengan 0, maka tampilkan B, jika tidak sama dengan 0, maka tampilkan A"

16. Penelusuran hasil fungsi logika dalam sel A2 yaitu:

=OR(AND(A1:B1);AND(B1:C1))

=OR(FALSE;FALSE)

Hasilnya FALSE

Penelusuran hasil fungsi logika dalam sel A3 yaitu:

=AND(OR(A1:B1);OR(B1:C1))

=AND(TRUE;FALSE)

Hasilnya FALSE

Penelusuran hasil fungsi logika dalam sel A4 yaitu:

=OR(AND(A1:B1);OR(A1:C1))

=OR(FALSE;TRUE)

Hasilnya TRUE

Perlu diingat bahwa:

OR akan menghasilkan TRUE bila melibatkan salah 1 pernyataan TRUE

OR akan menghasilkan FALSE bila semua pernyataan adalah FALSE

AND akan menghasilkan TRUE bila semua pernyataan adalah TRUE
