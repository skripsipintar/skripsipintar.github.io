Rumus present Continous Tense (Positive, Negative, Introgative) 3. Rumus There Are And There Is (Positive, Negative, Introgative) 4. Rumus Past Continous Tense (Positive, Negative, Introgative) 5. 1. Present Tense :

(+) S + V¹ (invinitive) + O

(–) S + do/does + not + V + O

(?) Do/does + S + V¹ + O + ? atau :

What/Who/Where/When/Why/How + do/does + S + V¹ + O + ?

Keterangan : (+) = kalimat positif / positive form(–) = kalimat negatif / negative form(?) = kalimat tanya / interrogative formS = subjek /subject (I,you,they,we,she,he,it)V¹ = kata kerja dasar (e.g. iron,eat,drink,etc)O = objek / objectDo = digunakan untuk subjek I,you,they,weDoes = digunakan untuk subjek she,he,it

2. Present Continous Tense

(+) S + to be + Ving(V progressive) + O

(–) S + to be + not + Ving + O

(?) to be + S + Ving + O + ? atau :

What/Who/Where/When/Why/How + to be + S + Ving + O + ?

Keterangan : (+) = kalimat positif / positive form(–) = kalimat negatif / negative form(?) = kalimat tanya / interrogative formS = subjek /subject (I,you,they,we,she,he,it)Ving = kata progressive / yang sedang berlangsung (e.g. drink →drinking,eat →eating,etc)O = objek / object to be = is,am,are

3. There is and There Are :

(+) There is + noun + Adverb

(–) There is + not + noun + Adverb

(?) Is there + noun + Adverb + ?

(+) There are + noun + Adverb

(–) There are + not + noun + Adverb

(?) Are there + noun + Adverb + ?

Keterangan : (+) = kalimat positif / positive form(–) = kalimat negatif / negative form(?) = kalimat tanya / interrogative form Adverb = kata keterangan noun = kata benda

4. Past Continous Tense :

(+) S + to be² + Ving(verb progressive) + O

(–) S + to be² + not + Ving + O

(?) to be² + S + Ving + O + ? atau :

What/Who/Where/When/Why/How + to be + S + Ving + O + ?

Keterangan : (+) = kalimat positif / positive form(–) = kalimat negatif / negative form(?) = kalimat tanya / interrogative formS = subjek /subject (I,you,they,we,she,he,it)Ving = kata progressive / yang sedang berlangsung (e.g. drink →drinking,eat →eating,etc)O = objek / object to be² = was,were

5. pronoun :

Subject pronouns : I, you, we, they, she, he, itObject pronouns : me, you, us, them, her, him, itPossessive pronouns : mine, yours, ours, theirs, hers, him, itsReflextive pronouns : myself, yourself/yourselves, ourselves, themselves, himself, herself, itselfDemonstrative pronouns : this --> these, that --> thoseIndefinite pronouns : somebody, nobody, anything, everything

Ada Contoh atau pengetahuan kata Simple Present Yaitu

Subjek = i, You, They, We, He, She, It

To Be = Am, are, are, are, is, is, is

Completed = I am Handsome, They Are Laty, You my Bestie Friend, We are in the hotel, He is my brother, She is Beautiful, It is Difficult

Bentuk Simple Present Tense

(+) Verbal = Subject + Verb. 1 Object

Nominal = Subject + to be, (is,am,are) + Object

Contoh 1 : They Study Litelature Every Monday

Contoh 2 : Mr.Joko Widodo is the president Indonesia

(-) Verbal = Subject + do not/does not + Verb.1 + Object

Nominal = Subject + to be not + Object

Contoh 1 : They do not Study Litelature Monday

Contoh 2 : Mr.Joko Widodo is not the President Indonesia

(?) Verbal = Do/Does + Subject + Verb.1 + Object

Nominal = To Be + Subject + Verb.1 Object

Contoh 1 : Do They Study Litelature Every Monday

Contoh 2 : Is Mr. Joko Widodo The President Indonesia?

37. Tolong buatlah 10 kalimat pronoun ( reflexive pronoun , demostrative pronoun , indefinite pronoun ) yg didalamnya ada kata adjective nya .... Tolong mau dikumpulkan besok senin ...


Kita menggunakan reflexive pronoun (kata ganti refleksif) saat kita ingin merujuk kembali ke subjek suatu kalimat atau klausa. Reflexive pronouns diakhiri dengan “-self” (tunggal) atau “-selves” (jamak).

Reflexive pronoun:

1. He sends a postcard to himself.

2. Help yourself.

3. The actors saved the local theatre money by making costumes themselves.

4. He wanted to impress her, so he baked a cake himself.

Baik pronoun maupun demonstrative adjective, memiliki parameter yang sama, yaitu jarak (distance) dan jumlah (number).

Berbeda dengan demonstrative adjective yang berfungsi menerangkan noun dan berada di depan noun yang diterangkan, demonstrative pronoun berfungsi mengantikan noun, baik yang berupa singular (this, that) atau plural (these, those). Pada klausa atau kalimat, singular noun diikuti singular verb dan berlaku sebaliknya.

Demostrative pronoun:

1. This is my car

2. That is clara's bike

3. These are his clothes

Demostrative adjective:

1. This bike is broken

2. That hat is expensive

3. These clothes are dry

Kata ganti tidak pasti (indefinite pronoun) adalah kata ganti yang mengacu pada makhluk, objek, atau tempat non-spesifik. Contoh kata ganti tidak pasti (indefinite pronoun): all, any, both, each, every, few, many, neither, nobody, and none.

Indefinite pronoun:

1. I like ice cream. I'll have some, please.

2. Some of the children are creating a raucous.

Indefinite adjective:

PEMBAHASANPronoun

Pronoun atau kata ganti adalah salah satu bagian dari part of speech. Pronoun berfungsi menggantikan kata asli dengan kata ganti agar kalimat dapat disampaikan dengan lebih padat dan minim pengulangan.

Personal Pronoun

Secara khusus, personal pronoun adalah kata ganti untuk noun atau kata benda. Contohnya manusia, hewan, lokasi, dan lain-lain. Kata ganti ini menyesuaikan fungsi kata pada kalimat (Subject atau Object)

1. SUBJECT PRONOUN

Berfungsi sebagai subjek/pelaku kegiatan.

I : SayaYou : AndaThey : MerekaWe : Kami, KitaHe : dia (laki-laki)She : dia (perempuan)It : dia (selain manusia)

2. OBJECT PRONOUN

Berfungsi sebagai objek/yang dikenai pelaku.

Me : SayaYou : AndaThem : MerekaUs : Kami, KitaHim : dia (laki-laki)Her : dia (perempuan)It : dia (selain manusia)DITANYAKAN
