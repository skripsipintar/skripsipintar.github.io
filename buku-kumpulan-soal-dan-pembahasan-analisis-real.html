1. Salah satu faktor dari 2x³ - 5x² - px + 3 adalah ( x + 1 ). Faktor lain dari suku banyak tersebut adalah...

A. (x - 2) dan (x - 3)

B. (x + 2) dan (2x - 1)

C. (2x - 1) dan (x - 3)

D. (2x + 1) dan (x - 2)

Pembahasan :

Karena x + 1 merupakan faktor suku banyak, maka suku banyak habis dibagi dan berlaku f(-1) = 0.

f(x) = 2x² - 5x² - px + 3 dari (x + 1) diperoleh x = -1

=> f(-1) = 2(-1)³ - 5(-1)² - p(-1) + 3

=> 0 = -2 - 5 + p + 3

=> 4 = p => substitusi nilai p = 4 ke fungsi suku banyak.

f(x) = 2x³ - 5x² - 4x + 3

=> f(x) = (x + 1) (2x² - 7x + 3)

=> f(x) = (x + 1) (2x - 1) (x - 3)

Jadi faktor lainnya adalah

C. (2x - 1) dan (x - 3)

1) metode(method)
Yaitu pemahaman terhadap material dsn bagaimana berinteraksi menjalin kepekaan melalui daya serap ,imajinasi,dan abstraksi agar dapat terjalin dari proses pembuatan melahirkan produk yg artistik.
2) asosiasi(association)
Yaitu kemampuan menghubungkan antar gagasan kemampuan panca indra dengan menggunakan bagan yg artistik.
3) estetika (aesthetice)
Merupakan ilmu keindahan yg diwujudkan dalam unsur desain
4) kebutuhan(need)
Yaitu desainer harus memiliki kepekaan yg tajam untuk memilih kebutuhan konsumen dan kemungkinan nya menjadi tren

Yaitu pemahaman fungsi yg mengubah desai dari suatu yg sifatnya personal menjadi komunal.
6) kegunaan(use)
Merupakan fungsi praktid dari sebuah desain dalam mewujudkan fungsi yg baik tentunya seotang desain harus mempertimbangkan siapa yg harus menggunakan.

Tentukan nilai a dan b jika x^3 - ax^2 + 5x + b habis dibagi x^2 - 2x - 3.

Pembahasan:

x^2 - 2x - 3 = (x - 3)(x + 1)

Disubstitusi kedalam x^3 - ax^2 + 5x + b = 0 menjadi :

f(3) = 3^3 - a(3)^2 + 5(3) + b = 0

0 = 42 - 9a + b

-42 = -9a + b……………(1)

f(-1) = (-1)^3 - a(-1)^2 + 5(-1) + b = 0

0 = -1 - a - 5 + b

6 = -a + b……………(2)

Dari persamaan (1) dan (2) diperoleh:

(-9a+b=-42) - (-a+b  =6)

a = 6      

b = 6 + a = 6 + 6 = 12


22. bagaimana pembuktian analisis real a.0=0


Anggap kita memiliki bilangan a dan b

Anggap teorema sudah dibuktikan bahwa untuk b bilangan real, maka berlaku :
0 + b = b

Sehingga,

a*(0 + b) = a*b
a(0) + a(b) = a(b)

Manipulasi nilai,

a(0) + a(b) - a(b) = a(b) - a(b)
a(0) = 0

Terbukti bahwa a(0) = 0
a*0 = 0

40. cari 2 soal kalimat di bikin type1 type 2 real type 3 real


Kelas: SMA
Pelajaran: English
Kategori: conditional
Pembahasan: 

Type 1 (possible event)
- If she calls me, I'll ask her.
- If she doesn't call me, I won't call her first.

Type 2 (imagination)
- If the alien attacked earth, I'd try to get to know it.
(the real: alien doesn't attack earth, so I won't try to get to know it)
- if North Korea started the war against USA, there would be a World War III. (the real: North Korea doesn't start the war yet, so there is no WW III.)

Type 3 (regrets)
- I would have sent it earlier, if I had known about it. 
(the real: I didn't know about it, so I wouldn't send it earlier)
- The murderer would have been caught, if the cops hadn't been fooled by him. (the real: the murderer wasn't caught because the cops were fooled by him)
