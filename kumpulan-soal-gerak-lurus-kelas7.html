7. 5 contoh soal tentang Massa Jenis Dan jawabannya KELAS7


Soal 1

Diketahui bahwa massa benda adalah 300 gram, setelah benda tersebut dicelupkan ke dalam larutan maka larutan bertambah 20[tex]cm^{3}[/tex]. Berapa massa jenis benda?

Massa jenis benda = [tex]\frac{massa}{volume} = \frac{300 gram}{20cm^{3} } = 15 \frac{g}{cm^{3} }[/tex]

Soal 2

Suatu logam aluminium mempunyai massa 120 gram dan volume 60 cm³ maka massa jenisnya adalah?

Massa jenis benda = [tex]\frac{massa}{volume} = \frac{120 gram}{60cm^{3} } = 2 \frac{g}{cm^{3} }[/tex]

Soal 3

Volume sebuah benda sebesar 9 cm³ jika massa jenisnya 14 g/cm³. Berapa besar massa benda tersebut?

Massa jenis benda = [tex]\frac{massa}{volume}[/tex]

Massa benda = massa jenis.volume

Massa benda = 14 g/cm³.9 cm³

Massa benda = 126 gram

Soal 4

Volume sebuah benda sebesar 20 cm³ jika massa jenisnya 12,5 g/cm³. Berapa besar massa benda tersebut?

Massa jenis benda = [tex]\frac{massa}{volume}[/tex]

Massa benda = massa jenis.volume

Massa benda = 12,5 g/cm³.20 cm³

Massa benda = 250 gram

Soal 5

Massa sebuah benda sebesar 1400 kg jika massa jenisnya 700 kg/m³. Berapa volume tersebut?

Massa jenis benda = [tex]\frac{massa}{volume}[/tex]

Volume = [tex]\frac{massa}{massa jenis benda}[/tex]

Volume = [tex]\frac{1400 kg}{700kg/m^{3} }[/tex]

Volume = 2m³

Pembahasan:

Massa jenis dapat diartikan sebagai kepadatan massa suatu benda per satuan volume. Rumus yang digunakan untuk mengukur massa jenis benda, yaitu:

Massa jenis benda = [tex]\frac{massa}{volume}[/tex]

18. temen temen tolong bikin soal aljabar kelas7 sama pembahasanya ini tugas dari guru saya​


Jawaban:

Berapakah hasil pemfaktoran dari bilangan berikut ini 16x2 − 9y2 ?

Jawabannya :

Harus di ingat bahwa bentuk faktor nya yakni aljabar :

a2 – b2 = ( a + b ) ( a – b )

16x2 = ( 4x )2

9y2 = ( 3y )2

Sehingga faktor dari bilangan 4x2 – 9y2 ialah ?

16x2 – 9y2 = ( 4x + 3y ) ( 4x – 3y )

Jadi, hasil pemfaktoran dari bilangan 16x2 − 9y2 ialah : ( 4x + 3y ) ( 4x – 3y ).

Soal No. 2

Tulislah bentuk sederhana dari bilangan berikut ini 2x2– 3x – 9 / 4x2 – 9 ?

Jawabannya :

Pemfaktoran dari pembilang nya :

2x2 – 3x – 9 = 2x2 – 6x + 3x – 9

= 2x ( x – 3 ) + 3 ( x -3 )

= ( 2x + 3 ) ( x – 3 )

Pemfaktoran dari penyebut nya :

4x2 – 9 = ( 2x – 3 ) ( 2x + 3 )

Sehingga akan diperoleh :

2x2 – 3x – 9 / 4x2 – 9 = ( 2x + 3 ) ( x – 3 ) / ( 2x – 3 ) ( 2x +3 )

Kemudian hilangkan faktor yang sama antara pembilang dan penyebut nya, yakni 2x + 3. Maka akan diperoleh sebuah hasil akhir seperti berikut ini :

2x2 – 3x – 9 / 4x2 – 9 = x -3 / 2x – 3

Jadi, hasil bentuk sederhana dari bilangan 2x2– 3x – 9 / 4x2 – 9 ialah : x -3 / 2x – 3.

Soal No. 3

Berapakah hasil dari bilangan berikut ini 2 ( 4x – 5 ) − 5x + 7 ?

Jawaban nya :

2 ( 4x 5 ) 5x + 7 = 8x -10 – 5x + 7

= 8x – 5x – 10 + 7

= 3x – 3

Jadi, hasil dari bilangan 2 ( 4x – 5 ) − 5x + 7 ialah : 3x – 3.

Soal No. 4

Tulislah bentuk sederhana dari bilangan berikut ini 6x2 + x – 2 / 4x2 – 1 ?

Jawaban nya :

Pemfaktoran dari pembilang nya :

6x2 + x – 2 = 6x2 – 3x + 4x – 2

= 3x ( 2x – 1 ) + 2 ( 2x – 1 )

= ( 3x + 2 ) ( 2x – 1 )

Pemfaktoran dari penyebut nya :

4x2 – 1 = ( 2x + 1 ) ( 2x – 1 )

Sehingga akan diperoleh :

6x2 + x – 2 / 4x2 – 1 = ( 3x + 2 ) ( 2x – 1 ) / ( 2x + 1 ) ( 2x – 1 )

Kemudian hilangkan faktor yang sama antara penyebut dan pembilang nya, yakni 2x – 1. Maka akan diperoleh sebuah hasil akhir seperti berikut ini :

6x2 + x – 2 / 4x2 – 1 = 3x + 2 / 2x + 1

Jadi, hasil bentuk sederhana dari bilangan 6x2 + x – 2 / 4x2 – 1 ialah : 3x + 2 / 2x + 1.

Soal No. 5

Berapakah hasil dari bilangan berikut ini ( 2x – 2 ) ( x + 5 ) ?

Jawaban nya :

( 2x – 2 ) ( x + 5 ) = 2x ( x + 5 ) – 2 ( x + 5 )

= 2x 2 + 10x – 2x – 10

= 2x 2 + 8x – 10

Jadi, hasil dari bilangan ( 2x – 2 ) ( x + 5 ) ialah : 2x 2 + 8x – 10.
