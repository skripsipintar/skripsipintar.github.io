23. tolong bantu ya soal ujian mandiri


ax² + 5x - 12 = 0

Salah satu akar persamaannya adalah 2, jadi:


(ax + .......) (x - 2) = 0 ⇒ salah satu akar persamaannya adalah 2 ( yaitu x-2)


Isian titik-titik tersebut adalah:


...... di kali -2 = -12 ⇒ jadi titik-titik tersebut adalah 6.


Jadi: (ax + 6) (x - 2) = 0


Untuk mencari nilai a adalah:


(ax dikali -2) + (6 dikali x) = 5x

-2ax + 6x = 5x

-2a + 6 = 5

-2a = -1

a = 1/2



Jadi ⇒ (1/2x + 6) (x - 2) = 0


Akar persamaan yang lainnya adalah:


(1/2x + 6) = 0

1/2x = -6

x = -12


Jawaban A

Diketahui:

P = 2021² - 2022 × 2020 + 2

Q = 2018 × 2020 - 2019²

Jawaban: A. P > Q

Penyelesaian:

P = 2021² - 2022 × 2020 + 2

= (2021 × 2021) - (2022 × 2020) + 2

= 4.084.441 - 4.084.440 + 2

= 1 + 2

= 3

Q = 2018 × 2020 - 2019²

= 4.076.360 - (2019 × 2019)

= 4.076.360 - 4.076.361

= -1

P = 3

Q = -1

3 lebih besar dari -1

Jadi, P > Q karena P (3) lebih besar dari Q (-1)
